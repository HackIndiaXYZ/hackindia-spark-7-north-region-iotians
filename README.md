# 🚛 Smart Fleet Intelligence — IoTians  
### HackIndia Spark 7 (North Region)

A modern **IoT-based fleet management system** that provides real-time tracking, driver insights, and intelligent decision-making for fleet owners.

---

## 🚀 Overview

This project combines:

- 📱 Flutter (Mobile App)
- ⚙️ Node.js + Express (Backend)
- 🔥 Firebase (Database & Auth)
- 🧠 Machine Learning (Driver Recommendation)

**Goal:**  
Make fleet operations **transparent, efficient, and data-driven**

## ✨ Features

### 👤 Fleet Owner
- Track all trucks in real-time  
- Monitor trips & drivers  
- Get ML-based driver recommendations  

### 🚛 Driver
- View assigned trips  
- See live truck data  
- Receive alerts  

### 🏭 Organization
- Track incoming/outgoing trucks  
- Monitor logistics flow  

---

## 🧠 ML Module

- Model: Linear Regression  
- Predicts: Driver Score  
- Output: Top driver recommendations  

**Features Used:**
- Safety score  
- Fuel efficiency  
- On-time delivery  
- Alerts  
- Experience

    flutter run

    cd fleet-backend
    npm install
    npm start
